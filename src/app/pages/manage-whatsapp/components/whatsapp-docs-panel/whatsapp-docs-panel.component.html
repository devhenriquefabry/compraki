Documentação WhatsApp

Resumo da arquitetura e operação do servidor multi-instância.

Segurança

As chamadas passam pelas Firebase Functions para proteger a chave da Evolution API.

Servidor Evolution API

A Evolution API roda fora do app e gerencia as conexões WhatsApp por instância.

Conexão por QR Code

Crie uma instância, gere o QR Code e leia pelo WhatsApp em aparelhos conectados.

Mensagem Teste

Use o botão de teste para validar se a instância conectada consegue enviar mensagens.

Guia completo: Consulte o arquivo docs/whatsapp-evolution-setup.md para deploy, variáveis e manutenção.
